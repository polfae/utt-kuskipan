# Úttøkukrøv

Ein einfald static heimasíða til at vísa úttøkukrøv í vektlyfting.

## Innihald

- `index.html` – bygnaðurin á síðuni
- `styles.css` – sniðgeving
- `data.js` – úttøkukrøvini
- `script.js` – tabs, stillingar, útrokningar og lyftara-kanning

## Funktiónir

- Vel kapping við tabs
- Sí menn og kvinnur í hvør sínari talvu
- Broyt lækkingarprosent í stillingum
- Lækkaðu úrslitini verða rundað til nærmasta heila tal
- Kanning har lyftari kann skriva kyn, føðiár, vektflokk og samlað úrslit
- Kappingar verða filtreraðar eftir aldri
- A-, B- og C-krøv kunnu vísast við undir-tabs, har tað er viðkomandi

## Brúk

Lat `index.html` upp í einum kaga, ella legg mappuna á GitHub Pages.
